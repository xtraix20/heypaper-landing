# HeYpaper - Landing Web Local Stationary 

## Description: 
HeY Paper is a landing web designed to a neighboorhood stationary in Soacha, the main pupose is increase the visibility online of the services and also increase the leads to specific clients like primary, high school and college studets, also mantaining some variety for their parents.

This project is planned like a simple, fastest and accesible solution for the local business without the digital presence.

## Problem
The business don't have: 
- Web page to explain the services or highly presence in the Internet
- A digital channel to quickly contact

## Solution 
Start a proposal of a Landing web page: 
- Bring a brief description of the services
- Make easier the contact by Whatsapp
- Responsive website for the mobile usage
- Use the owner domain to improve the credibility

## Implemented technologies
- HTML5
- CSS3
- Basic JavaScript
- Git
- Cloudfare DNS

## Features
- Responsive design
- Clearly structure using sections
- Services and products cards
